# Pricing Tiers Summary

Quick reference table for Shortly's pricing tiers with all key information.

---

## Free Tier

**Name**: Free

**Description**: Perfect for individual users and personal projects

**Annual Discount**: N/A (Always free)

**Monthly Base Fee**: $0

**Free Trial**: N/A (No trial needed)

**Features**:
- **Active Links (25 max)**: Maximum number of shortened links you can have active at one time
- **Custom Aliases (5/month)**: Create memorable short codes like "summer-sale" instead of random codes

---

## Starter Tier

**Name**: Starter

**Description**: Ideal for freelancers, content creators, and small businesses who need unlimited custom aliases and advanced analytics

**Annual Discount**: 17% off (equivalent to 2 months free)

**Monthly Base Fee**: 
- $9/month (billed monthly)
- $90/year ($7.50/month equivalent - save $18)

**Free Trial**: 14 days (full access to all Starter features)

**Features**:
- **Active Links (200)**: Create and maintain up to 200 active short links
- **Custom Aliases (Unlimited)**: Create unlimited memorable short codes with no monthly limit
- **Advanced Analytics**: Detailed insights including referrer sources, device types, and geographic locations
- **Custom Expiration Messages**: Personalize the message users see when links expire
